# 文章编写规则指南

本目录包含不同类型文章的编写规则 Prompt，用于指导 AI 或作者创作符合特定风格的文章。

## 文件说明

- `思考类文章.md` - 用于撰写观点性、分析性的思考文章
- `技术方案文章.md` - 用于撰写技术教程、实践指南类文章
- `产品介绍文章.md` - 用于撰写项目介绍、产品推广类文章

## 使用方法

在请求 AI 编写文章时，可以参考对应类型的编写规则，例如：

```
请根据 `.writing-guides/思考类文章.md` 中的规则，
编写一篇关于 XXX 的思考文章。
```

## 规则迭代

这些规则会随着写作实践不断优化。如果发现某种风格效果更好，可以及时更新对应的规则文件。
